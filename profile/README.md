## 主な特徴

- **Oracle 環境不要** - データベースやクライアントなしで .dmp ファイルの中身を直接閲覧
- **高速な解析エンジン** - C ネイティブ DLL による高速解析
- **2 つの DUMP 形式に対応** - EXP (レガシー) / EXPDP (DataPump)
- **高度な検索** - 12 種類の演算子、AND/OR 複合条件、大文字小文字区別
- **文字セット自動判定** - UTF-8 / Shift_JIS / EUC-JP を自動検出・変換
